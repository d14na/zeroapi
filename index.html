Code Example

The ZeroApi library is available to ALL Zeronet desktop and mobile clients by default, so as a developer you don't have to do anything extra.
You're welcome! 

/**
 * Perform basic user authentication.
 */
const userAuth = await ZeroApi.authorize(_username, _password, _crypto)

if (userAuth) {
    ZeroApi.navigate('/account/Main')
} else {
    ZeroApi.alert('authFailed')
}
